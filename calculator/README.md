# Standard Calculator

A basic calculator built using **HTML, CSS, and JavaScript**.

## Aim
To design a standard calculator interface and implement arithmetic operations using JavaScript.

## Features
- Addition, subtraction, multiplication, division
- Clear display function
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript

## Files
- index.html
- style.css
- script.js
